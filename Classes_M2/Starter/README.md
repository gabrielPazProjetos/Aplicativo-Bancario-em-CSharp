--- BankingApp (Classes_M2)
This is a simple banking application built in C#, developed to practice object-oriented programming concepts such as classes, properties, methods, encapsulation, and code organization.

--- Project Structure
BankCustomer.cs: Defines the BankCustomer class with properties and methods related to the customer.
BankAccount.cs: Defines the BankAccount class with operations such as deposit, withdrawal, transfer, and interest application.
Program.cs: Entry point of the program where objects are created and methods are demonstrated.
Classes_M2.csproj: Project configuration file for .NET.
.gitignore: Excludes unnecessary files and folders from version control.

--- Features
Create and update bank customers
Create bank accounts with initial balance and type
Deposit, withdraw, and transfer funds between accounts
Apply interest to account balances
Display customer and account information

--- Technologies
C#
.NET 8

Console application

--- Learning Objectives
Practice creating and using classes in C#
Apply encapsulation and good design practices
Understand how to organize a console project with multiple files
